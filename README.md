# Snap_Recipes_App
Projeto referente à 2ª Unidade da disciplina Programação para Dispositivos Móveis (PDM) do curso UFRN/TADS (4º Período). Será desenvolvido um app de receitas com a proposta de que vai detectar os ingredientes disponíveis através de fotos, listando e apresentando as receitas disponíveis.  Projeto desenvolvido em Dart com Flutter.

# App de Receitas com Reconhecimento de Ingredientes por Imagem
Um aplicativo que sugere receitas com base nos ingredientes disponíveis, utilizando visão computacional.
Requisitos Funcionais: 
1. Reconhecimento de Ingredientes 
o Identificar ingredientes por foto usando ML Kit ou TensorFlow Lite. 
2. Busca por Receitas 
o Sugerir receitas que utilizem os ingredientes identificados. 
3. Favoritos 
o Salvar receitas favoritas para acesso rápido. 
4. Modo Offline 
o Banco de dados local para buscar receitas offline. 
5. Compartilhamento 
o Compartilhar receitas via redes sociais ou PDF.
