# Snap_Recipes_App
Projeto referente à 2ª Unidade da disciplina Programação para Dispositivos Móveis (PDM) do curso UFRN/TADS (4º Período). Será desenvolvido um app de receitas com a proposta de que vai detectar os ingredientes disponíveis através de fotos, listando e apresentando as receitas disponíveis.  Projeto desenvolvido em Dart com Flutter.
